Dong Xu

I am an undergraduate researcher at Sun Yat-sen University, affiliated with SYSUSELab. My work focuses on LLM4SE, especially repository-level code intelligence, code graph construction, documentation generation, and training data synthesis for unseen codebases.

My current research explores how structured code representations, such as repository knowledge graphs and code dependency graphs, can help large language models understand, generate, document, and maintain software systems more reliably.

Research

  • Repository-level code intelligence: representing codebases as graphs for cross-file reasoning, semantic retrieval, and change impact analysis.
  • LLM-based software documentation: generating modular, cross-referenced, and incrementally maintainable documentation for large repositories.
  • Training for unseen codebases: synthesizing graph-grounded training data so models can adapt to newly released frameworks and private codebases.
