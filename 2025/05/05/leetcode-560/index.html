LeetCode 560: Subarray Sum Equals K

LeetCode 560

I solved this problem using C++.
My idea: use for loop to find the sum of subarrays, but this is not efficient.

It’s hard for me to find a better solution.
So I searched for it and found a solution using unordered_map to store the sum of subarrays.

This solution is more efficient than the previous one.

Explanation:
input: nums = [3, 4, 7, 2, -3, 1, 4, 2], k = 7
output: 4

prefixSumCount is a hash map that stores the count of prefix sums encountered so far.
prefixSum is the current prefix sum.

prefixSum[i] means the sum of nums[0] to nums[i].
prefixSum[i] = prefixSum[i-1] + nums[i]

so prefixSum[i-j] means the sum of nums[j] to nums[i].
If the sum of nums[j] to nums[i] is equal to k, then prefixSum[i] - prefixSum[j-1] = k.

so if we want to know how many subarrays have a sum equal to k, we can get it by checking how many times prefixSum[i] - k has appeared in the hash map.

so we can use unordered_map<int, int> to store the prefix sum and its count.
