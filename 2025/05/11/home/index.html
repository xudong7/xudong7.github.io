About Me

I am an undergraduate researcher at Sun Yat-sen University, working at the intersection of large language models and software engineering (LLM4SE). My research focuses on leveraging structured code representations — such as knowledge graphs — to enhance LLM-based code understanding, documentation generation, and repository-level reasoning.

Research Interests

  • LLM4SE — applying large language models to software engineering tasks
  • Code Intelligence — code representation learning, repository knowledge graphs, semantic code search
  • AI for Documentation — automated documentation generation, maintenance, and incremental update
  • Software Engineering — developer tools, program analysis, full-stack systems
