# Dong Xu

I am an undergraduate researcher at **Sun Yat-sen University**, affiliated with **SYSUSELab**. My research lies at the intersection of large language models and software engineering, with a focus on repository-level code intelligence, code graphs, and LLM-based software maintenance.

Currently, I am interested in using structured code representations, such as repository knowledge graphs and dependency-aware code graphs, to help language models understand, generate, document, and maintain large software systems.

## Research Interests

- Repository-level code intelligence and code graph construction
- LLM-based documentation generation and incremental maintenance
- Training data synthesis and adaptation for unseen codebases
- Software engineering tools powered by knowledge graphs and LLM agents

## Projects

- **RepoDoc**: knowledge graph based automatic documentation generation and incremental updates.
- **UCD-Training**: code-graph-grounded data synthesis and training for unseen codebases.
- **Repository Knowledge Graphs**: code entity extraction, dependency modeling, and semantic impact analysis.
